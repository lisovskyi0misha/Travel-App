# Travel-App
**Rails 7.0.4** \
**Ruby 3.1.2** \
**PosgreSQl 14.5** \
**Node 16.18.1** \
**Yarn**

## Project setup:
* `bundle install`
* `cp database.yml.sample database.yml` and setup with your data
* `rails db:create`
* `rails db:migrate`
* `yarn add esbuild`
* `yarn add @popperjs/core`
* `rails app:template LOCATION="https://railsbytes.com/script/xkjs12"`

## Running project:
`./bin/dev`
